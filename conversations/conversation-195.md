## Key Points
- why are all fonts on iphone smaller On iPhone, fonts can suddenly look smaller for a few common reasons: ### 1.
- **Display & Brightness / Text Size Settings** - Go to **Settings → Display & Brightness → Text Size**.
- - If the slider has moved left, system-wide fonts shrink.
- **Accessibility Zoom / Display Settings** - Check **Settings → Accessibility → Display & Text Size**.
- - Options like *Larger Text*, *Bold Text*, or *Reduce Transparency* can make fonts appear different.
- - If *Larger Accessibility Sizes* was off (or got reset after an update), text will look smaller.
